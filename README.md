Perfil de usuario objetivo

El proyecto está destinado a usuarios que valoran la estética visual, la variedad de estilos y la presentación clara de prendas inspiradas en tendencias de moda internacionales.

- Rango etario: Amplio, orientado a cualquier edad.
- Nacionalidad: Principalmente Argentina, con enfoque adaptable a público global.
- Ubicación geográfica: Contexto urbano, con orientación inicial a Resistencia, Chaco, pero accesible desde cualquier región.

Variables adicionales relevantes

- Nivel educativo: Sin restricción; estudiantes, profesionales y público general.
- Ocupación: Estudiantes, trabajadores, creadores de contenido y personas interesadas en la moda.
- Alfabetización digital: Media-alta; familiarizados con redes sociales y compras online.
- Dispositivos de acceso: Computadoras y dispositivos móviles (diseño responsive).
- Idioma principal: Español, con posibilidad de expansión futura.

Design Inspirations

- Global Fashion Platforms: Inspiración en tiendas de moda online reconocidas mundialmente, con catálogo visual y navegación clara.
- International Trends & Streetwear: Combinación de estilos internacionales (japonés, europeo y americano) para lograr una estética versátil.
- Minimalist E-commerce Design: Uso de tarjetas de productos, jerarquía visual clara y organización por categorías.
  
Creative Process Documentation

- Conceptualización: Se planteó una tienda de ropa basada en tendencias globales, sin limitarse a una sola estética, para atraer usuarios con distintos gustos.
- Decisiones estructurales: Se eligió una estructura centrada en catálogo visual, con categorías claras y navegación simple para facilitar la exploración.

Refinamiento visual:
Se descartó un enfoque exclusivamente japonés para ampliar la inspiración a moda internacional. Se incorporaron íconos por categorías, tarjetas de productos con efectos suaves y tipografía moderna para lograr una identidad visual clara y actual.
